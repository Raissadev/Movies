# Movies
<h3>Sobre o Projeto</h3>
<p>Sistema de filmes, com front-end UX/UI, usando o cURL para request da API do themoviedb.</p>

<ul>
  <li>Lenguages:</li>
  <i>PHP, JavaScript e SQL</i>
  
  <li>Database:</li>
  <i>MySql</i>
  
  <li>Style:</li>
  <i>Cascading Style Sheets</i>
  
  <li>Markup Language:</li>
  <i>HTML</i>
  
</ul>
